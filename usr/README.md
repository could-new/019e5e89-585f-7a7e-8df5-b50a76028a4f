# Shree Ram Application

A beautiful, serene Flutter application dedicated to Lord Ram. It features daily chaupais, a library of bhajans and chants, and a guide to the chapters of the Ramayana.

## Features
- **Daily Darshan:** View daily quotes and verses from the Ramcharitmanas.
- **Bhajans:** A beautifully designed list of popular bhajans and chants with mock playback support.
- **Wisdom:** An interactive timeline/expansion list of the 7 Kandas (books) of the Ramayana.
- **Cross-Platform:** Adaptive layout built to work flawlessly on Mobile, Tablet, Desktop, and Web.

## Tech Stack
- Framework: Flutter
- Language: Dart
- Architecture: Responsive UI with NavigationRail and NavigationBar depending on screen constraints.

## Setup
To run the app locally:
```bash
flutter pub get
flutter run
```
